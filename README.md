# TBankMultimodalLLMs
## Тестовое задание для совместной магистратуры МФТИ и ТБАНКА по направлению AI

### Как запустить этот код (На примере Kaggle)
- Добавьте в Kaggle все файлы .py как DataSet
- Добавьте в Kaggle файлы .pt как модели или DataSet (Для версионирования желательно как модели
- Запускайте ноутбуки
  - Qwen_Tests для кросс-валидации моделей Qwen на способность различать поставленные цели
  - Train-world-model для обучения мировой модели и теста на способность предсказывать будущее
  - Test-World-model для создания таблицы VLM оценок и её исследования, а также проверки модели на возмодность решать поставленные текстовые задачи.
